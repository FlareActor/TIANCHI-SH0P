## 基本思路：
1.类似摩拜赛，按照时间先后划分训练集和线下验证集，使用规则/模型构造负样本并训练分类器，对测试集同样构造候选集并选择置信度最大的作为结果（二分类）
2.每个mall构建一个分类器（多分类）
3.多分类结果构造候选，再送入二分类

1.多分类结果的概率信息加入二分类特征（提高1个百分点）
2.wifi_bssid Embedding/Hash Trick（失败）
